# EndPoint - API Rest

📌 **Autenticación**

|Método|Endpoint|Descripción|
|---|---|---|
|`POST`|`/api/login/`|Iniciar sesión|
|`POST`|`/api/logout/`|Cerrar sesión|

📌 **Gestión de Usuarios**

|Método|Endpoint|Descripción|
|---|---|---|
|`GET`|`/api/usuarios/`|Listar usuarios|
|`POST`|`/api/usuarios/`|Crear usuario|
